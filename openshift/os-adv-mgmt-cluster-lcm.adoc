= 集群生命週期管理
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
若要管理不同的 OpenShift 集群，您可以建立它們或將它們匯入進階集群管理。

. 首先導航到自動化基礎設施>叢集。
. 若要建立新的 OpenShift 集群，請完成以下步驟：
+
.. 建立提供者連接：導航至提供者連接並按一下新增連接，提供與所選提供者類型相對應的所有詳細信息，然後按一下新增。
+
image:redhat-openshift-075.png["新增提供者連接"]

.. 若要建立新集群，請導航至“集群”，然後按一下“新增集群”>“建立集群”。提供集群和相應提供者的詳細信息，然後按一下「建立」。
+
image:redhat-openshift-076.png["新增集群"]

.. 叢集建立完成後，出現在叢集清單中，狀態為Ready。


. 若要匯入現有集群，請完成以下步驟：
+
.. 導航至集群並點擊新增集群>匯入現有集群。
.. 輸入叢集名稱，點選儲存匯入並產生程式碼。顯示新增現有叢集的命令。
.. 按一下“複製命令”，在要新增到中心叢集的叢集上執行該命令。這將啟動叢集上必要代理的安裝，並且此過程完成後，叢集將以「就緒」狀態出現在叢集清單中。
+
image:redhat-openshift-077.png["導入現有集群"]



. 建立並匯入多個叢集後，您可以從單一控制台監控和管理它們。
