[role="lead"]
NetApp擁有多個符合Trident和Trident Protect 標準的儲存平台，可為容器化應用程式配置、保護和管理數據，從而有助於定義和最大化 DevOps 吞吐量。

[role="normal"]
NetApp擁有多個符合Trident的儲存平台，可為容器化應用程式設定、保護和管理資料。

image:redhat-openshift-043.png["此圖顯示輸入/輸出對話框或表示書面內容"]

* AFF和FAS系統運行NetApp ONTAP並為基於檔案 (NFS) 和基於區塊 (iSCSI) 的用例提供儲存。
* Cloud Volumes ONTAP和ONTAP Select分別在雲端和虛擬空間中提供相同的優勢。
* Google Cloud NetApp Volumes （AWS/GCP）和Azure NetApp Files在雲端提供基於檔案的儲存。



NOTE: NetApp產品組合中的每個儲存系統都可以簡化資料管理和內部站點與雲端之間的移動，以便您的資料位於您的應用程式所在的位置。
