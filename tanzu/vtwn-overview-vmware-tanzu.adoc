= VMware Tanzu 概述
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
VMware Tanzu 是一套產品組合，可協助企業實現其應用程式及其運行的基礎架構的現代化。  VMware Tanzu 的全端功能將開發和 IT 營運團隊整合到一個平台上，以便在本地和混合雲環境中一致地實現應用程式和基礎架構的現代化，從而持續向生產提供更好的軟體。

image:vtwn-001.png["VMware Tanzu 產品組合"]

若要詳細了解 Tanzu 產品組合中的不同產品及其功能，請造訪文檔link:https://docs.vmware.com/en/VMware-Tanzu/index.html["這裡"^]。

關於 Tanzu 的 Kubernetes 操作目錄，VMware 為 Tanzu Kubernetes Grid 提供了多種實現，所有這些實現都在各種平台上配置和管理 Tanzu Kubernetes 叢集的生命週期。  Tanzu Kubernetes 叢集是一個成熟的 Kubernetes 發行版，由 VMware 建置和支援。

NetApp已在其實驗室中測試並驗證了 VMware Tanzu 產品組合中以下產品的部署和互通性：

* link:vtwn-overview-tkg.html["VMware Tanzu Kubernetes 網格 (TKG)"]
* link:vtwn-overview-tkgs.html["VMware Tanzu Kubernetes 網格服務 (TKGS)"]
* link:vtwn-overview-tkgi.html["VMware Tanzu Kubernetes Grid 整合 (TKGI)"]
* link:vtwn-overview-vst.html["帶有 Tanzu 的 VMware vSphere（vSphere Pod）"]
